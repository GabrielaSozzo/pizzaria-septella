<h1 align="center">Pizzaria Septella</h1>


Este é um site institucional desenvolvido como trabalho colaborativo na ETEC, criado com HTML, CSS e JavaScript. O projeto simula o site de uma pizzaria, com foco em apresentar o cardápio, seções como "As mais pedidas", "Nossas unidades", "Localização", "Formulário para reservas" e "Contato" tudo isso com uma navegação fluida e agradável para o usuário.
